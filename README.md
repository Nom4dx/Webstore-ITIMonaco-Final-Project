### 🛍️ Webstore Project

> **Progetto scolastico** per la realizzazione di una webstore moderna e funzionale in PHP.

---

## 📖 Panoramica

Questo repository nasce come elaborato scolastico e ha come obiettivo la creazione di una **vetrina online** chiara e pulita. La struttura è volutamente leggera, così da rendere il progetto facile da comprendere, modificare e presentare durante le lezioni di Informatica.

## 🚀 Punti chiave

- 🎨 **Interfaccia Pulita**: Design semplice e leggibile.
- 🏗️ **Struttura Modulare**: Pronta per essere ampliata con database e carrello.
- 📚 **Approccio Didattico**: Codice commentato e logica lineare.
- 💻 **Base Ideale**: Ottimo punto di partenza per esami o progetti di gruppo.

## ⚙️ Setup Locale

Per testare il progetto basta un server PHP locale (es. XAMPP, WAMP o il server integrato).

### ⚡ Avvio Rapido

1.  **Clona il repository**:
    ```bash
    git clone https://github.com/disel/Webstore-ITIMonaco-Final-Project.git
    ```
2.  **Entra nella cartella**:
    ```bash
    cd Webstore-ITIMonaco-Final-Project
    ```
3.  **Avvia il server PHP**:
    ```bash
    php -S localhost:8000
    ```
4.  **Apri il browser**: Vai su `http://localhost:8000`
